Date: 19/01/2020

Auteur : CLEMENT Alexis

Description: Jeu de réflexion (inspiré du jeu "Démineur") dont le but est de localiser des mines cachées dans un champ virtuel avec pour seule indication le nombre de mines dans les zones adjacentes.

Contrôles:
   - Clic gauche: - Accès aux différentes difficultés
		  - Découvrir une case ou découvrir les cases autour d'un chiffre si le nombre de drapeaux est égal au chiffre
   - Clic droit: Poser ou retirer un drapeau
   - Espace: Rejouer une partie (même si elle n'est pas finie)
   - Echap: Quitter le jeu

Fonctionnalités: Pour gagner, le joueur doit découvrir toutes les cases non minées

Bugs connus:
   - Aucun
